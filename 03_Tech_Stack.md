# Suggested Tech Stack

## Frontend
- Flutter
- flutter_bloc
- dio
- go_router
- local_auth
- flutter_secure_storage
- fl_chart
- image_picker
- google_mlkit_text_recognition

## Backend Options

### Option 1 — ASP.NET Core
- ASP.NET Core Web API
- SQL Server
- Entity Framework Core
- JWT Authentication
- MonsterASP.NET hosting

### Option 2 — Firebase
- Firebase Authentication
- Firestore
- Firebase Storage

### Option 3 — Supabase
- Supabase Auth
- PostgreSQL
- Supabase Storage

## Important Note

Backend solution is NOT finalized yet.

Please analyze and recommend the best solution between:

- ASP.NET Core + MonsterASP.NET
- Firebase
- Supabase
- Hybrid architecture

The recommendation should consider:
- Scalability
- Cost (if there is a reliable free to pay option that will be good)
- Ease of development
- Analytics/reporting
- Family/shared wallets
- Long-term maintenance
