# Smart Expense & Family Finance Manager App

## Overview

This project is a cross-platform mobile application built using Flutter.

The goal of the app is to help users manage:

- Monthly income
- Expenses
- Family shared budgets
- Receipt scanning
- Expense analytics
- Remaining balance calculations
- Shared family wallets
- Spending reports and dashboards

The application should support:

- English and Arabic
- Light and Dark mode
- Fingerprint login
- Cloud synchronization
- Receipt OCR scanning
- Family/member collaboration

The application should be scalable because it will  later be released publicly on app store
